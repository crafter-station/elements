## Features

- Compatible with Vercel AI SDK
- 22+ popular models from 9 providers
- Streaming text with cursor animation
- Chat message components with role variants
- Token counting with progress visualization
- Temperature control with preset values
- Tool call display with status tracking
- Human-in-the-loop approval workflows
- Reasoning and chain-of-thought visualization
- Plan generation with streaming support
- Artifact containers for generated content
- Source citations with favicons
- AI debugging and inspection tools
- Multi-agent orchestration visualization
- Agent handoff and routing display
- Memory and context management
