## Overview

Custom authentication and user management components designed for [Clerk](https://clerk.com) Core 3. Built with the new redesigned hooks (`useSignIn()`, `useSignUp()`, `useWaitlist()`) that replace the deprecated Clerk Elements library.

These components are presentation-ready UI blocks that you install via shadcn and wire up with Clerk's hooks for real authentication.

## Setup

### 1. Install Clerk

```bash
bun add @clerk/nextjs
```

### 2. Get API Keys

From [dashboard.clerk.com](https://dashboard.clerk.com):

```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_...
CLERK_SECRET_KEY=sk_...
```

### 3. Wire Up Hooks

These components are presentation-ready. Connect them to Clerk's Core 3 hooks:

```tsx
import { useSignIn } from "@clerk/nextjs";
import { ClerkSignIn } from "@/components/elements/clerk-sign-in";

export function LoginPage() {
  const { signIn } = useSignIn();

  return (
    <ClerkSignIn
      onSubmit={async ({ email, password }) => {
        await signIn.password({ emailAddress: email, password });
      }}
    />
  );
}
```
