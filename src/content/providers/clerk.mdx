## Overview

Functional authentication and user management components built with [Clerk](https://clerk.com) Core 3 hooks. These components use `useSignIn()`, `useSignUp()`, `useWaitlist()`, `useUser()`, and `useOrganizationList()` directly — install via shadcn and they work out of the box.

## Setup

### 1. Install Clerk

```bash
bun add @clerk/nextjs
```

### 2. Get API Keys

From [dashboard.clerk.com](https://dashboard.clerk.com):

```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_...
CLERK_SECRET_KEY=sk_...
```

### 3. Use Components

Components are fully functional — just render them:

```tsx
import { ClerkSignIn } from "@/components/elements/clerk-sign-in";

export default function LoginPage() {
  return <ClerkSignIn redirectUrl="/dashboard" />;
}
```

For OAuth flows, add an SSO callback page:

```tsx
// app/sso-callback/page.tsx
import { AuthenticateWithRedirectCallback } from "@clerk/nextjs";

export default function SSOCallbackPage() {
  return <AuthenticateWithRedirectCallback />;
}
```
