## Installation

```bash
bunx shadcn@latest add @elements/json-viewer
bunx shadcn@latest add @elements/code-diff-viewer
bunx shadcn@latest add @elements/cli-output
bunx shadcn@latest add @elements/api-response-viewer
bunx shadcn@latest add @elements/error-boundary-ui
bunx shadcn@latest add @elements/env-editor
bunx shadcn@latest add @elements/schema-viewer
bunx shadcn@latest add @elements/webhook-tester
bunx shadcn@latest add @elements/og-image-explorer
```

## Features

- JSON tree viewer with search and copy path
- Code diff with Shiki syntax highlighting
- Terminal output with ANSI colors
- HTTP response viewer with timing
- Error boundary UI with stack traces
- Environment variable editor
- JSON Schema visualizer
- Webhook/API request tester
- OG image auditing tool
