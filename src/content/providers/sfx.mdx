## Overview

CC0-licensed sound effects distributed as both TypeScript modules (for React/Web Audio apps) and raw MP3 files (for video production and agents). All sounds are free for commercial use with no attribution required.

## Installation

### React / Web Audio

```bash
npx shadcn@latest add @elements/sfx-whoosh
```

This installs the sound as a TypeScript module with inline base64 audio, plus the `useSound` hook and sound engine.

### Video Production / Agents

```bash
curl -o public/sfx/whoosh.mp3 https://tryelements.dev/sfx/whoosh.mp3
```

### Discover All Sounds

```bash
curl -s https://tryelements.dev/r/sfx-index.json | jq '.sounds[].name'
```

## Usage

### React Hook

```tsx
import { useSound } from "@/hooks/use-sound";
import { whooshSound } from "@/sfx/whoosh";

export function TransitionButton() {
  const [play] = useSound(whooshSound, { volume: 0.5 });

  return <button onClick={play}>Transition</button>;
}
```

### Imperative Playback

```tsx
import { playSound } from "@/lib/sound-engine";
import { popSound } from "@/sfx/pop";

async function handleClick() {
  const { stop } = await playSound(popSound.dataUri, { volume: 0.3 });
}
```

### Video Production (Remotion)

```tsx
import { Audio, staticFile } from "remotion";

export function SfxLayer() {
  return <Audio src={staticFile("sfx/whoosh.mp3")} volume={0.5} />;
}
```

## Timing Guidelines

| Event | Sound | Timing |
|-------|-------|--------|
| Text appears | pop, click | 0ms (exact frame) |
| Scene transition | whoosh, swoosh | 100ms before cut |
| Fast cut | swoosh | 50ms before cut |
| Rewind/undo | reverse-whoosh | 100ms before |
| Zoom in | riser | 200ms before |
| Dramatic reveal | magic-reveal, boom | 50ms before |
| Photo-style cut | shutter | exact cut frame |
| Error state | error, glitch | at reveal |
| Success/deploy | success | at reveal |
| Build tension | drone | start of section |
| Hook emphasis | boom | exact frame |
| Notification | notification | at reveal |
