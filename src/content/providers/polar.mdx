Build beautiful monetization experiences with Polar. Display pricing cards, sponsor grids, subscription badges, revenue metrics, and license key management.

## Setup

### 1. Install Polar SDK (Optional)

```bash
npm install @polar-sh/sdk
```

### 2. Get API Keys

Get your Access Token from [polar.sh dashboard](https://polar.sh/dashboard).

### 3. Environment Variables

```bash
POLAR_ACCESS_TOKEN=polar_oat_...
```
