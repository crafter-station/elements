## Overview

A real cat meow recorded from a domestic cat. Perfect for playful notifications, attention hooks, and fun UI interactions. Duration: **0.77s**.

## Installation

### React / Web Audio

```bash
npx shadcn@latest add @elements/sfx-cat-meow
```

### Video Production / Agents

```bash
curl -o public/sfx/cat-meow.mp3 https://tryelements.dev/sfx/cat-meow.mp3
```

## Usage

### React Hook

```tsx
import { useSound } from "@/hooks/use-sound";
import { catMeowSound } from "@/sfx/cat-meow";

export function NotificationAlert() {
  const [play] = useSound(catMeowSound, { volume: 0.5 });

  return <button onClick={play}>Meow!</button>;
}
```

### Remotion

```tsx
import { Audio, staticFile } from "remotion";

export function CatMeowSfx() {
  return <Audio src={staticFile("sfx/cat-meow.mp3")} volume={0.4} />;
}
```

Play on notification events, attention prompts, or as a fun easter egg interaction.
