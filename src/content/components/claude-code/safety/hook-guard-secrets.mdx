## Installation

```bash
npx @tryelements/cli add-hook guard-secrets
```

## Event

**PreToolUse** with matcher `Edit` and `Write`. Checks `file_path` against a blocklist of sensitive file patterns.

## Blocked Files

`.env`, `.env.local`, `.env.production`, `credentials.json`, `service-account.json`, `id_rsa`, `id_ed25519`, `.pem`

## How It Works

Extracts the basename from `tool_input.file_path` and matches it against known sensitive file patterns. Any match returns exit code 2 to block the operation.

## Platforms

macOS, Linux.
