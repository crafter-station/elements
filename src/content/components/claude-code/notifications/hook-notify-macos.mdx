## Installation

```bash
npx @tryelements/cli add-hook notify-macos
```

## Event

**Notification** — fires when Claude needs human attention (idle, permission prompt, auth required).

## How It Works

Reads the hook event JSON from stdin, extracts `hook_event_name` and `notification_message`, then calls `osascript` to display a native macOS notification with the "Ping" sound.

## Platforms

macOS only (uses `osascript`). For cross-platform, use `notify-sound` instead.

## Source

```bash
#!/usr/bin/env bash
[[ "$(uname)" != "Darwin" ]] && exit 0

INPUT=$(cat)
TITLE=$(echo "$INPUT" | python3 -c \
  "import sys,json; d=json.load(sys.stdin); print(d.get('hook_event_name','Claude Code'))" \
  2>/dev/null || echo "Claude Code")
MESSAGE=$(echo "$INPUT" | python3 -c \
  "import sys,json; d=json.load(sys.stdin); print(d.get('notification_message', \
  d.get('hook_event_name','Needs attention')))" 2>/dev/null || echo "Needs attention")

osascript -e "display notification \"$MESSAGE\" with title \"Claude Code\" \
  subtitle \"$TITLE\" sound name \"Ping\""
```
