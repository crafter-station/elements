## Introduction

Welcome to Elements, a collection of full-stack components built by [Crafter Station](https://crafterstation.com). This documentation will help you understand and use the available components in your projects.

## Features

Elements provides a comprehensive set of features designed for modern React applications:

- **React components with TypeScript support** - Fully typed components for better development experience
- **Modern styling with Tailwind CSS** - Responsive and customizable designs
- **Authentication with Clerk** - Secure user authentication and management
- **Dark/light theme support** - Built-in theme switching capabilities
- **Full-stack integration** - Components that work seamlessly across your entire application

## Quick Start

Get started with Elements by installing the necessary dependencies and importing components:

```tsx
import { Button } from "@/components/ui/button";

export function Example() {
  return <Button>Click me!</Button>;
}
```
