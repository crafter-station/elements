## Installation

Install individual logos:

```bash
bunx shadcn@latest add https://tryelements.dev/r/apple-logo
```

Install the complete collection:

```bash
bunx shadcn@latest add https://tryelements.dev/r/logos
```

## Usage

```tsx
import { AppleLogo } from "@/registry/default/logos/apple";
import { GoogleLogo } from "@/registry/default/logos/google";

export function OAuthButtons() {
  return (
    <div className="space-y-2">
      <button className="flex items-center gap-2 px-4 py-2 border rounded">
        <AppleLogo className="h-4 w-4" />
        Continue with Apple
      </button>
      <button className="flex items-center gap-2 px-4 py-2 border rounded">
        <GoogleLogo className="h-4 w-4" />
        Continue with Google
      </button>
    </div>
  );
}
```

## Theme Adaptation

All logos automatically adapt to your theme:

- **Theme-aware logos** use `currentColor` and inherit text color
- **Brand logos** maintain official colors but adapt backgrounds
- **Scalable** at any size from 16px to 256px+
