# Apple Logo

A theme-adaptive Apple logo component that seamlessly integrates with your design system. Uses `currentColor` to automatically match your theme's foreground colors.

## Features

- **Theme Adaptive** - Automatically adapts to light and dark themes
- **Scalable Vector** - Crisp at any size using SVG
- **Design System Integration** - Uses `currentColor` for consistent styling
- **Accessibility** - Includes proper accessibility attributes
- **Customizable** - Easy to style with CSS classes and props

## Customization

### Size Variants

The logo scales proportionally with the provided dimensions:

```tsx
{/* Small */}
<AppleLogo className="h-4 w-4" />

{/* Medium */}  
<AppleLogo className="h-6 w-6" />

{/* Large */}
<AppleLogo className="h-8 w-8" />

{/* Extra Large */}
<AppleLogo className="h-12 w-12" />
```

### Color Customization

The logo uses `currentColor`, so it inherits the text color:

```tsx
{/* Inherits text color */}
<div className="text-blue-600">
  <AppleLogo className="h-6 w-6" />
</div>

{/* Custom color */}
<AppleLogo className="h-6 w-6 text-gray-600" />

{/* Theme-aware color */}
<AppleLogo className="h-6 w-6 text-primary" />
```

## Design Guidelines

### Apple Brand Guidelines

When using the Apple logo, follow Apple's brand guidelines:

- **Minimum Size** - Don't use smaller than 16px in digital applications
- **Clear Space** - Maintain adequate clear space around the logo
- **Color** - Use black on light backgrounds, white on dark backgrounds
- **Proportion** - Maintain the original aspect ratio

### Accessibility

The logo component includes proper accessibility attributes:

- `role="img"` for screen readers
- `aria-label` describing the logo
- Proper color contrast when using custom colors

## Theme Integration

The logo automatically adapts to your theme:

```css title="globals.css"
/* Light theme */
.apple-logo {
  color: #000000; /* Black on light backgrounds */
}

/* Dark theme */
.dark .apple-logo {
  color: #ffffff; /* White on dark backgrounds */
}
```
