# Clerk Waitlist (ShadCN)

Animated waitlist signup form with Clerk integration using custom ShadCN components. Requires ClerkProvider in root layout and waitlist mode enabled in Clerk dashboard (Configure > Restrictions > Sign-up mode).

## Preview

<div className="my-8 flex min-h-[400px] w-full items-center justify-center rounded-lg border bg-background p-8">
  <WaitlistElement />
</div>

**Features:** Smooth animations with motion effects, email validation and collection, waitlist position tracking, customizable success states, and seamless Clerk integration.

## Installation

```bash
bunx shadcn@latest add https://tryelements.dev/r/clerk-waitlist-shadcn
```

## Dependencies

```bash
npm install motion @clerk/backend @clerk/nextjs
```

## Environment Variables

```txt
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_
CLERK_SECRET_KEY=sk_test_
```

## Setup

1. **Enable waitlist mode** in Clerk Dashboard:
   - Go to Configure > Restrictions > Sign-up mode
   - Select "Waitlist" or "Restricted"

2. **Configure ClerkProvider**:

```tsx title="app/layout.tsx"
import { ClerkProvider } from "@clerk/nextjs";

export default function RootLayout({ children }) {
  return (
    <ClerkProvider>
      <html lang="en">
        <body>{children}</body>
      </html>
    </ClerkProvider>
  );
}
```
