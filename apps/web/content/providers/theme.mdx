## Overview

A collection of theme switcher components that allow users to toggle between light and dark modes. Each variant offers different UI patterns and interactions, from simple switches to multi-button controls with system preference support.

## Installation

Install any theme switcher variant individually using the commands shown above, or install all variants:

```bash
bunx shadcn@latest add @elements/theme-switcher
bunx shadcn@latest add @elements/theme-switcher-switch
bunx shadcn@latest add @elements/theme-switcher-button
bunx shadcn@latest add @elements/theme-switcher-dropdown
bunx shadcn@latest add @elements/theme-switcher-toggle
bunx shadcn@latest add @elements/theme-switcher-multi-button
```

## Setup Required

All theme switcher components require the `next-themes` ThemeProvider to be set up in your root layout:

```tsx
import { ThemeProvider } from "next-themes";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

**Important:** Add `suppressHydrationWarning` to your `<html>` tag to prevent hydration warnings.

## Usage

Import and use any theme switcher variant in your application:

```tsx
import { ThemeSwitcher } from "@/components/elements/theme-switcher";
import { ThemeSwitcherButton } from "@/components/elements/theme-switcher-button";
import { ThemeSwitcherDropdown } from "@/components/elements/theme-switcher-dropdown";

export default function Header() {
  return (
    <header className="flex items-center justify-between p-4">
      <h1>My App</h1>
      {/* Use any variant */}
      <ThemeSwitcher />
    </header>
  );
}
```

## Features

- **Multiple Variants** - Choose from 6 different theme switcher styles
- **Smooth Animations** - All variants include polished transitions
- **System Preference** - Dropdown and Multi Button support system theme
- **Type Safe** - Built with TypeScript for better DX
- **Accessible** - WCAG compliant with screen reader support
- **Loading States** - Built-in skeleton states during hydration
